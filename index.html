Domestic Publications

  • Jinseok Kim, Sooyeon Hwang, Sukmin Cho, Soyeong Jeong, Sungzoon Cho,
    “Synergistic Query Reformulation via LLMs and MLM-based Query Encoding in SPLADE”, Under Review
    ➤ Analyzes how different SPLADE query encoders (MLM vs. MLP) behave when LLM-based query expansion is applied. It finds that the synergy between surface-level expansion (via LLMs) and internal reformulation (via MLM) leads to significant performance gains.
    My Role: First author – Idea proposal, experiment design, model training, and analysis.
  • Sooyeon Hwang, Jinseok Kim, Gahgene Gweon,
    “An Object-aware Auxiliary Module for Reference-free Image Caption Evaluation with a Large Multimodal Model”, Under Review
    ➤ Proposes an object-aware auxiliary module that enhances reference-free image caption evaluation by incorporating object-level visual information. By combining cropped object images and grounding-based score adjustment, the method improves alignment with human judgment without requiring additional training.
    My Role: Co-author – Proposed the idea of using cropped object images for fine-grained visual alignment.
  • Jinseok Kim, Sukmin Cho, Soyeong Jeong, Jong C. Park,
    “Controllable prompt tuning with relation dependent tokens”, Proceedings of Korea Computer Congress (KCC 2023), Jeju
    ➤ Proposes a unified prompt encoder that incorporates relation-dependent control tokens, allowing a single model to generate prompts adaptable to diverse knowledge probing tasks.
